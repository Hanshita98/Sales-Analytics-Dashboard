# Sales-Analytics-Dashboard# 📊 Sales Analytics Dashboard

A Business Intelligence Dashboard built using Python, Streamlit and Plotly.

## Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Region Filter
- Monthly Sales Trend
- Sales by Region
- Product Distribution Pie Chart
- Download CSV Report
- Top Products Table

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
